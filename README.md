# WizDrobe

Mobile wardrobe app for organizing clothing items and planning outfits.

## Setup

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed
- A supported device target:
  - Chrome/Edge for web preview, or
  - Android emulator/device, or
  - iOS simulator/device (macOS)
- Git (optional, for version control)

### Install
```bash
flutter pub get
```

### Run
```bash
flutter run
```

Optional device-specific commands:

```bash
flutter run -d chrome
flutter run -d edge
```
